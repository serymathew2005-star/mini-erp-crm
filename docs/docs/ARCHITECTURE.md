# System Architecture

## Overview

Mini ERP + CRM is a full-stack web application developed using React, Node.js, Express, Prisma ORM, and SQLite.

## Architecture

Frontend (React + Vite)
        ↓
REST API (Express)
        ↓
JWT Authentication
        ↓
Prisma ORM
        ↓
SQLite Database

## Modules

- Authentication
- Dashboard
- Customer CRM
- Product Management
- Inventory Management
- Sales Challan Management
- Stock Movement Logs
