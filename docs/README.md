# Mini ERP + CRM Documentation

## Project Overview
Mini ERP + CRM Operations Portal developed using React, Node.js, Express, Prisma ORM, and SQLite.

## Technology Stack
- React + Vite
- Node.js + Express
- Prisma ORM
- SQLite
- JWT Authentication
